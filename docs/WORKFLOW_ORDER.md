# Workflow Order

Run scripts from the repository root after configuring authorised local data paths. The workflow mirrors the manuscript emphasis on original non-isolated versus ID-isolated cross-wave evaluation, Logistic A-only versus XGBoost A+B, accuracy-sensitivity discordance, calibration, decision utility, and strict-label robustness.

## 1. Data preparation

1. `python scripts/core/01_import_qc.py`
2. `python scripts/core/02_define_cohort_label.py`
3. `python scripts/core/03_finalize_possible_sarcopenia_label.py`
4. `python scripts/core/04_variable_dictionary_and_missingness_audit.py`
5. `python scripts/core/05_build_full_analysis_base.py`
6. `python scripts/core/21_audit_wave_overlap_and_build_id_isolated_split.py`
7. `python scripts/core/21b_prepare_model_input_id_isolated.py`

## 2. Primary model development

1. `python scripts/evaluation/21c_model_logistic_id_isolated.py`
2. `python scripts/evaluation/21d_model_xgboost_id_isolated.py`

## 3. Original and ID-isolated cross-wave evaluation

1. `python scripts/evaluation/21e_temporal_validation_id_isolated.py`
2. `python scripts/evaluation/27_generate_table2_main_model_performance_id_isolated.py`
3. `python scripts/evaluation/26_generate_table3_figure2_calibration_id_isolated.py`

The script names retain historical `validation` wording, but the manuscript and public documentation describe the present analysis as cross-wave evaluation and ID-isolated evaluation. Five-fold cross-validation terminology is preserved for Wave 1 internal performance estimation.

## 4. Thresholds, feature importance, and decision curves

1. `python scripts/sensitivity/23_screening_threshold_analysis.py`
2. `python scripts/sensitivity/01_extract_coefficients_and_importance_v1.py`
3. `python scripts/sensitivity/02_generate_dca_net_benefit_v1.py`
4. `python scripts/sensitivity/03_generate_sensitivity_summary_tables_v1.py`

## 5. Strict-label and v2 robustness analyses

1. `python scripts/sensitivity/01_strict_label_sensitivity_v2.py`
2. `python scripts/sensitivity/02_generate_discrimination_calibration_v2.py`
3. `python scripts/sensitivity/03_generate_paired_model_comparison_v2.py`
4. `python scripts/sensitivity/04_generate_subgroup_robustness_v2.py`
5. `python scripts/sensitivity/05_generate_calibration_uncertainty_v2.py`
6. `python scripts/sensitivity/06_generate_submission_figures_tables_v2.py`

## 6. Figures and tables

1. `python scripts/figures/24_generate_figure1_participant_flow.py`
2. Final Figure 2 and Figure 4 are generated by v2 scripts listed above.
3. Figure 3 decision curves are generated by `scripts/sensitivity/02_generate_dca_net_benefit_v1.py`.

## 7. Checks

1. `python scripts/checks/04_check_submission_enhancement_outputs_v1.py`
2. `python scripts/sensitivity/08_check_submission_enhancement_v2.py`
3. `python scripts/sensitivity/09_check_v9_final_consistency.py`
4. `python tools/check_public_release_v1.py`

Do not run the full workflow unless authorised CHARLS data are available locally.
