# Data Access

This release does not include CHARLS participant-level data. Researchers must apply for authorised access through the official CHARLS portal and follow all data-use restrictions.

Do not commit raw CHARLS files, derived row-level analysis files, model-input tables, row-level prediction files, trained models, or participant identifiers to this repository.

The public aggregate examples in `example_outputs/` are provided only to demonstrate final manuscript outputs and are not a substitute for authorised data access.

## Manuscript Context

This release supports participant-isolated cross-wave evaluation of possible sarcopenia screening models in CHARLS. It does not include data needed to rerun participant-level analyses.
